## The split that keeps it honest

| | contract (`assert_backend_conforms`) | external judge (`caps.independent`) |
|---|---|---|
| `MemoryBackend` (mock) | ✅ same callable | ❌ refused — proves *mechanics* |
| `OpenObserveBackend` | ✅ same callable | ✅ carried — proves *arrival* |

A mock that passed a *different* contract would be drift waiting to happen; a
mock that claimed *independence* would be a false judge. This receipt pins both
halves, with Longinus anchors (`# KG: contract-mock-parity-receipt-2026-07-22`)
in `backends/conformance.py`, `backends/memory.py`, the test, this script, and
this document.
